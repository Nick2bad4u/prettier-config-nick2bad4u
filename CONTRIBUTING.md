# Contributing

Thanks for helping improve `prettier-config-nick2bad4u`.

## Development

1. Install dependencies with `npm install`.
2. Make source changes in `src/prettier.config.mts`, `src/preset.mts`, tests, fixtures, or docs.
3. Run `npm run build` after source changes so `dist/` package artifacts are updated.
4. Run `npm run verify` before opening a pull request.

## Pull requests

- Keep changes focused.
- Explain why formatting rules or plugin dependency changes are needed.
- Include compatibility notes when changing plugin-backed file overrides.
